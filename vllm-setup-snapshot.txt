============================================================
CRITICAL vLLM FLAGS (for this model)
============================================================
--trust-remote-code          Required for Nemotron custom code
--mamba_ssm_cache_dtype float32   Mamba2 hybrid layer cache precision
--tool-call-parser nemotron_json  Custom tool call parser
--reasoning-parser nemotron_nano_v2  Enables thinking/reasoning output
--enable-auto-tool-choice    Auto tool selection

Note: --enable-prefix-caching was used in nemotron-vllm.service
(currently inactive) but NOT in the gateway config. The gateway
version (without prefix caching) is what produced the RAG results.
